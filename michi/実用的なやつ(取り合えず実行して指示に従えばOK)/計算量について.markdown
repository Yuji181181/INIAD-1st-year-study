## 計算量
O(1) < O(logn) < O(n) < O(nlogn) < O(n^2) < O(C^n)
* 計算時間が複数の項の和になるとき、最も早く増加する項のみ残す。
* 計算時間が定数倍されても、オーダーは変わらない。

liner_search（線形探索）の計算量:O(n)
binary_search（二分探索）の計算量:O(logn)

マージソートの計算量
平均時間計算量:O(nlogn)
最悪時間計算量:O(nlogn)

選択ソートの計算量
平均時間計算量:O(n^2)
最悪時間計算量:O(n^2)

クイックソートの計算量
平均時間計算量:O(n^2)
最悪時間計算量:O(nlogn)
