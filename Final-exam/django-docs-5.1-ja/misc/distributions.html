サードパーティの Django ディストリビューション¶

多くのサードパーティのディストリビュータが、パッケージ管理システムと統合した Django のバージョンを提供するようになりました。これらのシステムは、Django が必要とする依存関係 (データベースアダプタなど) を自動的にインストールする機能を含んでいるので、Django のユーザにとってインストールやアップグレードをより簡単にすることができます。

通常、これらのパッケージは最新の安定版 Django をベースにしているので、開発版 Django を使いたい場合は、私たちの Git リポジトリから 開発中のバージョンをインストールする の指示に従う必要があります。

Linux や OpenSolaris のような Unixプラットフォームを使っているなら、ディストリビュータが Django を既にパッケージ化しているかどうか確認してください。もしあなたが Linux を使っていて、パッケージが利用可能かどうかを調べる方法を知らないのなら、今が勉強する良い機会です。 Django Wiki には Third Party Distributions のリストがあり、あなたを助けてくれるでしょう。

ディストリビュータ向け¶

Django をディストリビューションにパッケージングしたい場合、喜んで手助けします！ django-developers メーリングリストに参加して、自己紹介をしてください。

すべてのディストリビュータは django-announce メーリングリストを購読することを推奨しています。Django の新しいリリースや重要なバグフィックスをお知らせするための (非常に) わずかのメールしか送られないメーリングリストです。
