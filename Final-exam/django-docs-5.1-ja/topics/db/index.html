モデルとデータベース¶

モデルは、データに関する唯一かつ決定的な情報源です。あなたが保持するデータが必要とするフィールドとその動作を定義します。一般的に、各モデルは単一のデータベースのテーブルに対応付けられます。

  • 検索
  • マネージャ
  • 素の SQL 文の実行
  • データベースのトランザクション
  • 複数のデータベース
  • テーブル空間（tablespace）
