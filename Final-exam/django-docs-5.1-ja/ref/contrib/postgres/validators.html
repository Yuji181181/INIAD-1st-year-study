バリデータ (Validator)¶

これらのバリデータは django.contrib.postgres.validators モジュールから利用可能です。

KeysValidator¶

class KeysValidator(keys, strict=False, messages=None)[ソース]¶

与えられたキーが値に含まれていることを検証します。もし strict が True の場合、他にキーが存在しないことも確認します。

渡される messages は、missing_keys および/または extra_keys を含む辞書である必要があります。

注釈

与えられたキーの存在のみを確認し、キーの値が空でないことを確認しない点に留意してください。

範囲バリデータ¶

RangeMaxValueValidator¶

class RangeMaxValueValidator(limit_value, message=None)[ソース]¶

範囲の上限が limit_value を超えていないことを検証します。

RangeMinValueValidator¶

class RangeMinValueValidator(limit_value, message=None)[ソース]¶

範囲の下限が limit_value より小さくないことを検証します。
