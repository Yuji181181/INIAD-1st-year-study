
# 木構造について

- **木構造**は、ノードとそれらのノードの間の接続（エッジ）からなるデータ構造です。
- 各ノードは、0個以上の子ノードを持つことができます。
- 木構造は、**連結していて閉路（ループ）がない**無向グラフの一種です。
- 主な特性：
  - **根ノード**: 木の最上部にあるノード。
  - **葉ノード**: 子ノードを持たないノード。
  
# Pythonでの木構造の実装

```python
class Node:
    def __init__(self, value):
        self.value = value  # ノードの値
        self.left = None    # 左の子ノード
        self.right = None   # 右の子ノード

# ノードの作成
root = Node(1)
root.left = Node(2)
root.right = Node(3)
```
二分木の探索について木を探索するためには、次の3つの方法があります。行きがけ順（Preorder Traversal）
手順:

このノードを訪れる（＝表示する）
左の子ノードを根とする部分木を探索する
右の子ノードを根とする部分木を探索する


訪問順序: 根 → 左 → 右
通りがけ順（Inorder Traversal）
手順:

左の子ノードを頂点とする部分木を探索する
このノードを訪れる（＝表示する）
右の子ノードを頂点とする部分木を探索する


訪問順序: 左 → 根 → 右
帰りがけ順（Postorder Traversal）
手順:

左の子ノードを頂点とする部分木を探索する
右の子ノードを頂点とする部分木を探索する
このノードを訪れる（＝表示する）


訪問順序: 左 → 右 → 根
ノードの高さ、木の高さ、ノードの深さノードの深さ
定義: ノードの深さは、そのノードまでの経路に含まれるエッジの数（親ノードからの距離）です。
例: 根ノードの深さは0で、根ノードから1つ下のレベルのノード（子ノード）の深さは1、というように、各レベルを1ずつ増加します。
ノードの高さ
定義: ノードの高さは、そのノードから葉ノードへ至る最長経路のエッジの数です。
例: 葉ノードの高さは0で、親ノードは子ノードの高さの最大値に1を加えたものです。
計算方法:

ノードが葉ノードの場合、高さは0。
それ以外の場合、ノードの高さは「左の子ノードの高さ」と「右の子ノードの高さ」の大きい方に1を加えたものです。


木の高さ
定義: 木の高さは、根ノードから最も深い葉ノードまでの最長経路のエッジの数です。
例: 木の全てのノードの高さを考慮して、根ノードが持つ高さを木の高さとします。
